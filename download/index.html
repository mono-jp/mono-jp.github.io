Mono for Linux is available for various distributions

Please refer to the installation guide for more information about how to install and configure your Mono environment.

Xamarin packages

These packages are built, tested and distributed by Xamarin. Use these if you want to use a stable, official and up-to-date version of Mono.

  • Debian, Ubuntu, and derivatives
  • CentOS, Fedora, and derivatives
  • openSUSE and SLES

Linux distribution packages

Mono is packaged in various Linux distributions by third-party maintainers. Some packages do not ship the latest Mono version; this depends on the distribution/maintainer.

  • Arch Linux
  • CentOS EPEL
  • Debian
  • Fedora
  • Gentoo
  • openSUSE
  • Ubuntu

Continuous Integration packages

Every commit to our repository automatically builds distribution packages. Use these if you want a bleeding edge Mono version and don't care about stability.

Learn more about CI packages

Community packages

These packages are provided by the community and linked here for your convenience. They are not tested by Xamarin, so use them at your own risk.

  • Mono Packages installed to /opt for Debian and Ubuntu
  • Mono Packages installed to /opt for CentOS
  • Latest Fedora mono packages for Fedora Rawhide and earlier Fedora versions
  • Latest SUSE mono packages, part of the official "Factory" repository
  • Latest Slackware mono packages from SlackPack repository
