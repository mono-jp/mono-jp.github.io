Cross platform, open source .NET framework

Monoは開発者の皆さんが簡単にクロスプラットフォーム アプリケーションを作成できるように設計されたソフトウェアであり、 .NET Foundationに属します。

Monoは、Microsoftがスポンサーとなって、 C#とCommon Language Runtimeという ECMA標準に基づく、Microsoftの.NET Frameworkをオープンソースで実装したものです。 発展するソリューション ビジネスの一群と、アクティブで熱心に協力してくれるコミュニティが、Monoの地位を、クロスプラットフォーム アプリケーションのリーディング チョイスとなるところまで、押し上げてくれました。
